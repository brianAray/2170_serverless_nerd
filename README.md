# Welcome to 2170 Serverless NERD (Node.js, Express.js, React, DynamoDB)!

This is the training repository for the Serverless NERD cohort starting on September 2nd, 2025. All of the notes, demos, study guides, project guidelines, and other resources will be found here.

## Installation Guide

For training, we will need the following:

- [Git](https://git-scm.com/download)
- [Visual Studio Code](https://code.visualstudio.com)
- [Node LTS](https://nodejs.org/en)
- [Postman](https://www.postman.com)

You will also need to setup a free tier AWS Account, here is a guide on how:

- [AWS Account Setup](https://repost.aws/knowledge-center/create-and-activate-aws-account)

The rest of the technologies will be covered during training.
